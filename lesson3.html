Lesson 3: Adding CSS Style and HTML Structure

CSS Styling

A typical CSS syntax looks like this: selector {property: value;}
A selector can be an HTML element, or a particular chosen part of the document, called a class. A class name will have a dot in front of it, like this: .classname
The part of the syntax in the curly braces is called a declaration. It has a property, which is a particular style attribute, such as a color or a size, and a value for that attribute.

There are three ways to add CSS styling to an HTML page:
  1. Write CSS code in the head element of HTML. The element <style> is used.
  2. Link to a separate CSS file. Within the head element of the HTML is a link to the file, which will have this structure: <link rel="stylesheet" href="style.css">
  3. Write CSS inline within the HTML document within the <div> tags. This is not advised because it would result in repetitive coding, which could potentially cause problems when editing, and is generally a waste of time.

Box Sizing and Positioning

HTML elements are boxes with 4 components: margin, border, padding, and content. To better control the sizing of a box, use percentages instead of pixels, and use box-sizing: border-box. You can also specify properties for specific browsers by using the prefixes -moz (Firefox), -ms (Internet Explorer), or -webkit (other browsers).

<div> is a block element, which by default, takes up the whole width of a page. Using display: flex; allows <div> elements to display side by side.

Code, Test, Refine

Generally, create a page based on a design as follows:
  1. Look for natural boxes (groupings)
  2. Look for repeated styles and semantic elements
  3. Write the HTML
  4. Apply CSS styles
  5. Repeat steps 3 and 4 until the page looks like the design.
