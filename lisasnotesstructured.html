Lesson 1: Getting Started with HTML

World Wide Web Summary

The World Wide Web (WWW) is a collection of documents that use Hypertext Markup Language (HTML). Webpages are connected by hyperlinks (links). Files can be found on the web include HTML, plain text, images, videos, and music.

A web browser on a computer (such as Internet Explorer) sends a request to servers through the Internet, using Hypertext Transfer Protocol (HTTP), and the servers respond by sending files that can be viewed or played by the browser.

HTML

Basic Components

HTML is made up of:

  • Text content
  • Markup (formatting of the content)
  • References to other documents, such as images
  • Links to other pages

Tags

HTML markup uses tags, which are typically structured as <tagname>content</tagname>. The first tag is called an "opening tag," the second tag (with the slash) is called a "closing tag." The entire structure is called an element.

Examples of tags with this structure are <b>bold</b> and <em>emphasis (italic)</em>

Tags can also have attributes, which equal some kind of value. The structure of a tag with an attribute is <tagname attribute="value">content</tagname> An example of a tag with an attribute is the anchor tag <a>, which is used to create links. A tag like this one
<a href="http://www.ledorestaurant.com/">Pizza for lunch!</a> would display like this in the browser:
Pizza for lunch!

An image tag is an example of a void tag, in which the tag contains no content, and thus does not need a closing tag. An image tag also contains two attributes: src, whose value is the link to the image, and alt, whose value is text that describes the image for Section 508 compliance. This tag <img src="https://upload.wikimedia.org/wikipedia/commons/0/00/Copo_de_nieve.png" alt="snowflake"> would display like this:

There are a number of tags that can group and format content:

  • <br> break
  • <p> paragraph
  • <span> span
  • <div> div

These tags all contain content, and therefore use closing tags. <br> and <span> are inline tags, meaning their formatting is done within a line. <p> and <div> are block tags, meaning that they form a block, or box of content. <span> and <div> can also control the style of their content.

Document Structure

A basic HTML document has these components:

  • Doctype <!DOCTYPE HTML>
  • HTML tags at the beginning and the end of the document
  • Head tags, which includes relevant metadata, CSS, and Javascript functions
  • Body tags, which contain the content of the document

This class will mostly focus on the body of the document.

Lesson 2: Creating a Structured Document

General Page Structure

A web page generally has a "tree-like structure," or hierarchy. A <div>, for example, can have within it <h1> and <p> tags, which in turn can have text and tags within them. This is also called a Document Object Model (DOM).

HTML is the language that controls the structure of the webpage. CSS is the language that controls the visual style of the page. Different parts of the page, such as images, text, and headers, can be placed on the pages like boxes. Putting things in boxes (by using the <div> tag), is the first step towards controlling the style of a page.

Lesson 3: Adding CSS Style and HTML Structure

CSS Styling

A typical CSS syntax looks like this: selector {property: value;}
A selector can be an HTML element, or a particular chosen part of the document, called a class. A class name will have a dot in front of it, like this: .classname
The part of the syntax in the curly braces is called a declaration. It has a property, which is a particular style attribute, such as a color or a size, and a value for that attribute.

There are three ways to add CSS styling to an HTML page:
  1. Write CSS code in the head element of HTML. The element <style> is used.
  2. Link to a separate CSS file. Within the head element of the HTML is a link to the file, which will have this structure: <link rel="stylesheet" href="sheetname.css">
  3. Write CSS inline within the HTML document within the <div> tags. This is not advised because it would result in repetitive coding, which could potentially cause problems when editing, and is generally a waste of time.

Box Sizing and Positioning

HTML elements are boxes with 4 components: margin, border, padding, and content. To better control the sizing of a box, use percentages instead of pixels, and use box-sizing: border-box. You can also specify properties for specific browsers by using the prefixes -moz (Firefox), -ms (Internet Explorer), or -webkit (other browsers).

<div> is a block element, which by default, takes up the whole width of a page. Using display: flex; allows <div> elements to display side by side.

Code, Test, Refine

Generally, create a page based on a design as follows:
  1. Look for natural boxes (groupings)
  2. Look for repeated styles and semantic elements
  3. Write the HTML
  4. Apply CSS styles
  5. Repeat steps 3 and 4 until the page looks like the design
