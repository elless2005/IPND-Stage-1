Lesson 2: Creating a Structured Document

General Page Structure

A web page generally has a "tree-like structure," or hierarchy. A <div>, for example, can have within it <h1> and <p> tags, which in turn can have text and tags within them. This is also called a Document Object Model (DOM).

HTML is the language that controls the structure of the webpage. CSS is the language that controls the visual style of the page. Different parts of the page, such as images, text, and headers, can be placed on the pages like boxes. Putting things in boxes (by using the <div> tag), is the first step towards controlling the style of a page.
